# Image Editor App

A simple and lightweight web-based Image Editor built using JavaScript and HTML Canvas.  
This project allows users to upload an image, apply basic editing operations, and preview changes in real time.

The app is designed for learning purposes and demonstrates how image manipulation works in the browser without using external libraries.

---

## Features

- Upload images from local system
- Adjust brightness, contrast, and saturation
- Apply filters like grayscale, sepia, blur, and invert
- Rotate and flip images
- Real-time preview using HTML Canvas
- Reset image to original state
- Download edited image

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- HTML Canvas API
